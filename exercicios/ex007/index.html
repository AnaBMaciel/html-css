Principais Formatações

Abaixo apresento a você as principais formatações existentes em HTML5

Texto em negrito

O texto em negrito é usado para dar destaque a uma palavra ou frase. Para isso, usamos a tag STRONG

Texto em itálico

O texto em itálico é usado para dar ênfase a uma palavra ou frase. Para isso, usamos a tag EM

Texto marcado

O texto marcado é usado para destacar uma parte do texto. Para isso, usamos a tag MARK

Texto pequeno

O texto pequeno é usado para diminuir o tamanho da fonte. Para isso, usamos a tag SMALL

Texto inserido

O texto inserido é usado para mostrar um texto que foi inserido. Para isso, usamos a tag INS

Texto deletado

O texto deletado é usado para mostrar um texto que foi deletado. Para isso, usamos a tag DEL

Texto subscrito

O texto subscrito é usado para mostrar um texto abaixo da linha. Para isso, usamos a tag SUB

Texto sobrescrito

O texto sobrescrito é usado para mostrar um texto acima da linha. Para isso, usamos a tag SUP
